# App

Application developed in **React-Native**

### Start android

    npm run android

### Start IOS

    npm run ios

### Start without cache

1. On a tab

       npm run start:reset

2. In another tab

       npm run android
       // or
       npm run ios

### Clean cache (android)

    npm run clean

### Build apk for installation in the device (android)

    npm run build:apk

### Build aap for publication (android)

    npm run build:aap
